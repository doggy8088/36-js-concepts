簡介

這個專案是為了幫助開發者掌握 JavaScript 概念而建立的。它不是必備，但在未來學習 JavaScript 的過程中，可以作為一篇指南。本篇文章是基於 Stephen Curtis 的文章寫成，你可以從這裡閱讀。

原文地址: https://github.com/leonardomso/33-js-concepts
